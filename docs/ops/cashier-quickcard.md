# Cashier Quick Card

Print this and tape it next to the POS terminal.

---

## Starting your shift

1. Open **POS** → enter your 4-6 digit **PIN**.
2. **Open Cash Drawer** → count starting float → enter amounts → confirm.

## Ringing up a sale

- **Scan barcode** (or type it) → item appears in Current Sale.
- Type to search a name if no barcode.
- **+ / –** buttons change quantity. **✕** removes an item.
- **F8** or click **Pay Cash** → tap a quick-tender button (EUR 10 / 20 / 50 / 100) or type the amount → **Confirm**. Change is shown.
- Click **Pay Card** → follow the prompt on the card terminal.

## Mistakes

- **F4** voids the current (not-yet-paid) sale.
- **Storno last sale** (red button) — voids the *last completed* sale. Use immediately; don't leave the screen first.
- **Reprint receipt** — prints the last transaction again if paper jammed or customer wants another copy.

## Returns

- **F9** or **Returns** button → scan the original receipt → pick items → confirm.

## Paid in / Paid out

- Pay supplier cash, take cash out, deposit change: use **Paid In / Paid Out** in the kassenbuch — always write a reason.

## Ending your shift

1. **Close Cash Drawer** → count every denomination → enter amounts.
2. System shows expected vs counted → difference is recorded.
3. Z-Report (manager only) → end-of-day totals.

## Trouble

- **Receipt printer offline** → red dot top-right. Finish the sale anyway (TSE still signs), then **Reprint receipt** once paper is back.
- **Card terminal offline** → red dot. Take cash, or ask the customer to wait.
- **Nothing works** → call the owner. Do NOT reboot the NUC mid-shift.
